BookTracker

Книжный трекер с возможность писать заметки, а также получать достижения за свои книжные результаты.

Для пользователей:

Основные функции: 
1. Возможность отслеживать время своего чтения при помощи таймера
2. Возможность создавать заметки к любой странице книги 
3. Ведение небольшой статистики чтения и библиотеки

Как установить:

1. Откройте ссылку https://github.com/AgataSergeevaPR31/KP_18_Sergeeva_BookTracker/releases/download/v1.0.0/app-release.apk
2. Нажмите "Скачать"
3. Найдите скачанный файл в приложении "Файлы" или "Загрузки"
4. Дайте разрешение на установку из неизвестных источников
5. Нажмите "Установить", дождитесь окончания установки и нажмите "Готово" или "Открыть"

Для разработчиков:
Язык: Kotlin
Минимальная SDK: API 24 (Android 7.0)
Архитектура: MVVM 
Библиотеки: ViewBinding, Room, Coroutines, Hilt, WorkManager и др.
