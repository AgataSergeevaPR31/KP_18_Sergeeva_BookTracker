BookTracker

Книжный трекер с возможность писать заметки, а также получать достижения за свои книжные результаты.

Для пользователей:

Основные функции: 
1. Возможность отслеживать время своего чтения при помощи таймера
2. Возможность создавать заметки к любой странице книги 
3. Ведение небольшой статистики чтения и библиотеки

Как установить:

1. Перейдите в раздел ..\app\build\outputs\apk\debug
2. Скачайте последний файл app-debug.apk
3. Установите его на ваше Android-устройство.
4. (Если нужно) Разрешите установку из неизвестных источников.

Для разработчиков:
Язык: Kotlin
Минимальная SDK: API 24 (Android 7.0)
Архитектура: MVVM 
Библиотеки: ViewBinding, Room, Coroutines, Hilt, WorkManager и др.
